Services

GEO DISCOVERY SERVICE

Hosts a dataset of moving resources. This service looks up and returns the collection of the resources in specific proximity of provided location.

Go to API

GEO TRACKING SERVICE

Stores the location of the moving resource and allow subscription to the tracked resource's location. This service also allows creating and saving trip records.

Go to API

GEO INFORMATION SERVICE

Hosts map data. This data represents objects such as roads, forests, canals, railway tracks as well as virtual things like administrative boundaries and geofenses.
